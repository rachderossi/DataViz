## Dispersão - Microsoft Excel

Após abrir a base de dados no Microsoft Excel:

- Seleione as duas colunas que são do seu interesse para criar o gráfico de dispersão

- Clique em Inserir > Gráfico
<img src="https://github.com/rachderossi/DataViz/assets/64082110/ce7eb30c-2871-44e2-9260-ab3c6a892302" width = "300" height ="600">

- Na aba Configuração escolha Dispersão
- É possível clicar na aba Personalizar ao lado de Configuração para editar o nome dos eixos, título do gráfico e valores máximos e mínimos do eixo horizontal e vertical
<img src="https://github.com/rachderossi/DataViz/assets/64082110/5012b9ba-c5ff-4d5a-811e-a594739f8858" width = "300" height ="600">

- O gráfico deve se parecer com esse
<img src="https://github.com/rachderossi/DataViz/assets/64082110/9b70f5c3-16f4-417a-b68c-11d1ed92ca88" width = "700" height ="600">

## Dispersão - Looker Studio

Após abrir o looker studio no seu navegador de preferência:

- Clique em Relatório em branco
<img width="1040" alt="image" src="https://github.com/rachderossi/DataViz/assets/64082110/d500679b-a633-47dd-82e6-906dd3857bc9">

- Clique em Adicionar dados e escolha a conexão de sua preferência
<img width="1228" alt="image" src="https://github.com/rachderossi/DataViz/assets/64082110/752ccefc-ce0f-4281-8fae-50bc75dd7d2a">

- Na aba Configuaração araste para o campo Dimensão as duas variáveis do seu interesse para criar o gráfico de dispersão
- Arraste as mesmas variáveis nos campos Métrica X e Métrica Y
- É possível clicar na aba Estilo ao lado de Configuaração para editar o nome dos eixos, título do gráfico e valores máximos e mínimos do eixo horizontal e vertical
<img width="491" alt="image" src="https://github.com/rachderossi/DataViz/assets/64082110/5e7f2040-4fa1-41d5-80e6-ffa8cfad781b">

- No botão acima de Configuraçâo escolha o gráfico do tipo dispersão
