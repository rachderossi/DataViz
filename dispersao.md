## Dispersão - Microsoft Excel

Após abrir a base de dados no Microsoft Excel:

- Seleione as duas colunas que são do seu interesse para criar o gráfico de Dispersão

- Clique em Inserir > Gráfico
<img src="https://github.com/rachderossi/DataViz/assets/64082110/ce7eb30c-2871-44e2-9260-ab3c6a892302" width = "300" height ="600">

- Na aba Configuração escolha Dispersão
- É possível clicar na aba Personalizar ao lado de Configuração para editar o nome dos eixos, título do gráfico e valores máximos e mínimos do eixo horizontal e vertical
<img src="https://github.com/rachderossi/DataViz/assets/64082110/5012b9ba-c5ff-4d5a-811e-a594739f8858" width = "300" height ="600">

- O gráfico deve se parecer com esse
<img src="https://github.com/rachderossi/DataViz/assets/64082110/9b70f5c3-16f4-417a-b68c-11d1ed92ca88" width = "800" height ="600">

## Dispersão - Looker Studio

Após abrir o Looker Studio no seu navegador de preferência:

- Clique em Relatório em branco
<img src="https://github.com/rachderossi/DataViz/assets/64082110/d500679b-a633-47dd-82e6-906dd3857bc9" width = "800" height ="500">

- Clique em Adicionar dados e escolha a conexão de sua preferência
<img src="https://github.com/rachderossi/DataViz/assets/64082110/752ccefc-ce0f-4281-8fae-50bc75dd7d2a" width = "900" height ="600">

- Na aba Configuração araste para o campo Dimensão as duas variáveis do seu interesse para criar o gráfico de Dispersão
- Arraste as mesmas variáveis nos campos Métrica X e Métrica Y
- É possível clicar na aba Estilo ao lado de Configuração para editar o nome dos eixos, título do gráfico e valores máximos e mínimos do eixo horizontal e vertical
<img src="https://github.com/rachderossi/DataViz/assets/64082110/5e7f2040-4fa1-41d5-80e6-ffa8cfad781b" width = "500" height ="500">

- No botão acima de Configuração escolha o gráfico do tipo Dispersão
<img src="https://github.com/rachderossi/DataViz/assets/64082110/290c72b4-d762-4567-b8a4-5d79283dc4ad" width = "500" height ="500">

- O gráfico deve se parecer com esse
<img src="https://github.com/rachderossi/DataViz/assets/64082110/6ff4d247-aead-49a0-91e3-28123260efe6" width = "800" height ="600">
